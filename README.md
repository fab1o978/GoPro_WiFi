# GoPro_WiFi (HERO3+ Black only)
Arduino Library to remote control a GoPro Camera using a WiFi connection (ESP8266)

**Warning!!! This is still in early alpha, not usable yet**

## Functions ##
- [x] Detect Camera Power State
- [x] Camera Power On
- [x] Camera Power Off
- [x] Take Picture
- [ ] Start/Stop Video
- [ ] Mode Change
- [ ] Camera Informations
